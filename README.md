# Cadastro de Usuários

Projeto simples de cadastro de usuários desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades
- Cadastrar usuários
- Listar usuários
- Editar usuários
- Excluir usuários
- Armazenar dados no localStorage

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript

## Objetivo
Projeto criado para praticar manipulação do DOM, eventos, formulários e armazenamento local no navegador.
